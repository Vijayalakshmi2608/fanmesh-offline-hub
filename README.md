# FanMesh AI

**The Offline Football Super App**

Stadium-grade fan super app featuring nearby mesh chat, AI translation, marketplace, wallet, and emergency alerts that work offline.

## Technologies Used

- React 19
- TypeScript
- TanStack Router & TanStack Start
- Tailwind CSS v4
- shadcn/ui & Radix UI primitives
- Framer Motion

## Getting Started

### Prerequisites

Ensure you have [Bun](https://bun.sh) installed.

### Installation

Clone the repository and install dependencies:

```bash
bun install
```

### Development

Run the development server locally:

```bash
bun run dev
```

### Building for Production

Create a production build:

```bash
bun run build
```

## Features

- **Nearby Mesh Chat:** Communicate with fans around you without an internet connection.
- **AI Translation:** Break down language barriers with built-in real-time AI translation.
- **Marketplace:** Trade and purchase fan gear and tickets securely.
- **Wallet:** Manage your digital assets and transactions natively.
- **Emergency Alerts:** Stay safe with critical broadcast alerts that work even when offline.
